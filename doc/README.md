## Parts of speech

FN (noun): Ex. SARA, LUIS, ...
LN (noun): Ex. GARCIA, RODRIGUEZ, ...
PRE (preposition): [DA, DE, DI, DO, DU, DAS, DEL, DES, DOS, VAN, VON]
ART (article): [AL, DEN, DER, LA, LAS, LOS]
CON (conjunction): [I, Y]
PAT (patronymic): [BEN, EL, MAC, MC, O, SAN, ST]
